# 🌱 Telegram-бот «Агроном» – AI-помощник по уходу за растениями

Бот диагностирует болезни растений по фотографии листа с помощью нейросети (InceptionResNetV2, обученной на датасете PlantVillage) и даёт рекомендации на русском языке.

## 🚀 Возможности
- Распознавание 38 классов болезней и здоровых растений
- Поддержка культур: томаты, картофель, яблоня, виноград, кукуруза, перец, клубника и др.
- Подробные рекомендации по лечению
- Ответы на русском языке
